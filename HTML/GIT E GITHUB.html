O GIT E GITHUB

Repositorio de Programadores

AMBOS:

Se tratam de Repositórios para guardar seus códigos!

GIT:

Aplicação desenvolvida pela Linux. Se trata de um Repositório apenas local (Salvo no dispositivo); Criado para o programador salvar todas as versões do código para o desfazer alguma alteração, caso queira. A grande desvantagem é que se a maquiagena pifar perde tudo.

GITHUB:

Aplicação desenvolvida pela Microsoft. Muito melhor do que o GIT; Pois o amarzenamento é remoto (Na nuvem). Além da comunidade de Programadores, você pode criar seu portfólio.
