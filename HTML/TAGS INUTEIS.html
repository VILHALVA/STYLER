Sim, é verdade! Existem sim algumas Tags que são tão inuteis no HTML quanto uma escova para um careca 🌚!

O code por exemplo, é util apenas para destacar a fonte de algum codigo, ou seja, apenas estetica. (Eu achando que servia para executar algum codigo no site 😑)

Já o q é usado para colocar apenas ASPAS. É mais facil digitar essa tag, do que colocar aspas na frase, isso é pra evitar a fadiga 🤤.

O blockquote teoricamente, serve para citar frases prontas, na pratica é a mesma coisa do code.

E a mais inutil de todas é o bdo, ele apenas Exibe a frase ao contrario, Isso é util para ajudar os hebreus chegar na terra prometida!
